<b>About</b><br/>
ViViScreen is the first Dapp of Groo Corporation.<br />

<b>Getting Started</b><br/>
1)Clone project<br/>
2)Open with Android Studio

```
com.android.tools.build:gradle:2.3.3
```

```
19
```

- ``` 'io.socket:socket.io-client:0.8.3'```  - Socket IO Client
